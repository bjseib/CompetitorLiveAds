Fetch live ads from the Meta Graph API, filter by RMG category and publisher, and keep screenshots handy for quick reviews.

Quick start

  1. Paste a valid Graph API access token.
  2. Select your RMG categories and publishers.
  3. Run the search to see grouped ads with snapshots.

All publishers are restricted to the United States ad region.

Filters

Active filters appear above the results. Use include/exclude to narrow down categories.

RMG category
Pick a category then choose whether to include or exclude it from results.
Result limit Applies to each request sent to the Meta Graph API.

Publishers

Select one or more publishers. Results will be grouped and screenshot links added for quick reference.

Ads are always requested for the United States region to match publisher requirements.
